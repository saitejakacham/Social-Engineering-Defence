# Social Engineering Defense

This repository contains the PowerPoint presentation submitted as part of the **Security Awareness Training Design Project** under the **PG Certificate Program in AI / GenAI Powered Cybersecurity**.

The project focuses on understanding social engineering attacks and designing an effective security awareness training program to reduce human-based security risks.

## Topics Covered
- Social Engineering Fundamentals  
- Pretexting, Baiting, Tailgating & Psychological Manipulation  
- Training Needs Analysis  
- Module Design and Assessment Strategy  
- Engagement Techniques and Implementation Plan
